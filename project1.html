News Feed Analysis Website

Description:
This Java website takes the URL of an RSS file (it was https://news.yahoo.com/rss in the video) and parses it for article information (including title, date, source, and URL). Then, the program processes the HTML file of each news article and isolates the main text. It separates the text into sentences and analyzes each sentences' sentiment (via Stanford Core NLP). The average of all sentence sentiment values is calculated and displayed on the website. Additionally, I added parallel processing and real-time updates to enhance performance.

Inspiration:
I made this website because I was tired of reading negative news articles. I wanted to know how positive or negative an article was before reading it.

What I Learned:
I learned a lot about Natural Language Processing and data parsing. I also learned about parallel systems and got better at optimizing code.
