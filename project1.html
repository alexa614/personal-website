News Feed Analysis Website

Description:
This Java website takes the url of an RSS file ((LINK yahoo.rss?????) was used in the video), parses it for article information (including title, date, source, and link). Then the program processes the html file of each news article and isolates the main text of the article. This text is then broken up into sentences and each sentence's sentiment is analyzed using Stanford Core NLP. The sentiment value of each sentence is averaged to get a final overal sentiment of the article, which is displayed on the website. Additionally, to reduce the runtime, I used paralell processing so four threads were running concurrently (so four articles can be processed at once).

Inspiration:
I got the idea for this project because I read a lot of negative news articles and I wanted a way to find a happy news story. I made this website because I was tired of reading tons of negative news articles. I wanted to know how positive or negative an article was going to be before reading it. This website does it.

What I Learned:
While making this project, I learned a lot about Natural Language Processing and data parsing. I also learned about paralell systems and got better at optimizing code.
